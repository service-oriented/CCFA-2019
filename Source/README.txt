trong cấu trúc trên:

Các package:

hudala.mvc.controller: Chứa các servlet class thực hiện nhận request từ người dùng và trả về view tương ứng (nhiệm vụ điều hướng).
hudala.mvc.model.dao: Chứa các class thực hiện nhiệm vụ tương tác với cơ sở dữ liệu.
hudala.mvc.model,bean: Chứa các class định nghĩa cho đối tượng, các class này sẽ được ánh xạ đến các bảng trong cơ sở dữ liệu.
hudala.mvc.model.service: Chứa các class có nhiệm vụ thực hiện các business logic (xử lý nghiệp vụ).
hudal.mvc.common: Chứa các class được định nghĩa để dùng như là các thư viện.
Các folder:

view: Chứa các file jsp 
images: chứa các ảnh
